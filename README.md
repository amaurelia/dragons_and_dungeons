# Dragons and Dungeons

## Descripción del proyecto
Es un proyecto que tiene APIS REST para el backend que obtienen la información de una base de datos, la cual agrega esta información en una página HTML creada para jugar en línea.

## Cómo ejecutar el proyecto

### Pre Requisitos
* Tener instalado NodeJs
* Tener las credenciales de una Base de datos

### Instalación del proyecto
Ejecutamos npm install

### Ejecución del proyecto
Ejecutamos npm start
